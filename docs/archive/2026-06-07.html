Critical Everest Forms Pro flaw exploited to take over WordPress sites

Hackers are exploiting a critical vulnerability in Everest Forms Pro, potentially compromising WordPress sites and exposing sensitive data.

Who should care: WordPress administrators and security teams managing sites with the Everest Forms Pro plugin should be particularly vigilant.
Attack path: Attackers can exploit the vulnerability to execute arbitrary code on the server, effectively taking control of the WordPress site. This can lead to data breaches or further compromise of the hosting environment.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: A serious flaw in the Everest Forms Pro plugin is being actively exploited, allowing attackers to take over WordPress sites.
cve wordpress plugin
  • A critical vulnerability (CVE-2026-3300) in the Everest Forms Pro plugin has been identified and is currently being exploited by hackers. This flaw allows attackers to gain complete control over affected WordPress sites.
  • The exploitation of this vulnerability can lead to unauthorized access to sensitive data, site defacement, or further attacks on connected systems. Organizations using this plugin must act quickly to mitigate risks and protect their assets.
  • Attackers can exploit the vulnerability to execute arbitrary code on the server, effectively taking control of the WordPress site. This can lead to data breaches or further compromise of the hosting environment.
  • Watch next: Monitor for updates from the plugin vendor regarding patches and any reports of increased exploitation attempts.
