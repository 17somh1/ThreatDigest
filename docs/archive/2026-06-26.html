Who should care: E-commerce teams and customer support departments using the Shop app are most exposed.
Attack path: Attackers leverage the Shop app's legitimate interface to create a sense of trust, making it easier to deceive users into taking harmful actions.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: Threat actors are using the Shop app to send fake purchase receipts, tricking users into revealing sensitive information.
phishing shopify e commerce

Attack path: Attackers gain leverage by tricking hotel staff into opening malicious ZIP files, which then deploy the Node.js implant to access internal systems.
Why this matters now: Useful context for teams tracking where attacker pressure is building.

phishing node.js hospitality
