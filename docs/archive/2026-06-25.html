Malicious Edge extension abuses Native Messaging as bridge to malware

The exploitation of a malicious Edge extension poses a significant risk of ransomware deployment, potentially compromising sensitive systems.

Who should care: IT security teams managing Microsoft Edge environments and any organizations using this browser.
Attack path: Attackers leverage the Native Messaging API to execute malicious code outside the browser's sandbox, facilitating the installation of ransomware.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: A malicious Edge extension is being used to bypass security measures and deploy ransomware.
ransomware cve cloud
  • A malicious Microsoft Edge extension named 'Edgecution' has been identified as a tool used in a ransomware attack. This extension abuses the Native Messaging API to escape the browser's sandbox, allowing it to execute a Python-based backdoor on infected systems.
  • The ability to bypass browser security measures increases the risk of widespread ransomware infections. Organizations using Microsoft Edge should be particularly vigilant, as this attack vector could lead to significant data loss and operational disruption.
  • Attackers leverage the Native Messaging API to execute malicious code outside the browser's sandbox, facilitating the installation of ransomware.
  • Watch next: Monitor for updates from Microsoft regarding patches or mitigations for this vulnerability.
Key terms
  • Native Messaging - a feature that allows web applications to communicate with native applications installed on the user's computer.
  • Ransomware - a type of malware that encrypts files on a victim's system, demanding payment for decryption.
Defender focus
  • Implement monitoring for unauthorized Edge extensions in your environment.
  • Review and restrict the use of Native Messaging APIs where possible.
  • Conduct a security audit of systems using Microsoft Edge to identify potential vulnerabilities.
Do now
  • Immediately review installed Edge extensions for any unauthorized or suspicious entries.
