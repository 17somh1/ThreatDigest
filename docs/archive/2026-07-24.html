Bing Images Flaws Let Crafted SVGs Run Commands as SYSTEM on Microsoft's Servers

The exploitation of Bing's image processing flaws could allow attackers to execute arbitrary commands on Microsoft's servers, posing a significant risk to data integrity and service availability.

Who should care: Development, security, and operations teams working with Bing's image processing services should prioritize this issue.
Attack path: An attacker could submit a specially crafted SVG file to Bing's image search, leveraging the vulnerability to execute commands with elevated privileges on the underlying servers.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: Bing's image processing has critical vulnerabilities that allow crafted SVG files to run commands with elevated privileges.
cve bing image processing
  • A crafted SVG file submitted to Bing's image search was able to execute commands as NT AUTHORITY\SYSTEM on Microsoft's production image-processing workers and as root on Linux machines. Testing by XBOW confirmed that this vulnerability affected multiple workers across different hosts and network ranges, indicating a systemic issue within Bing's image processing tier. Microsoft has issued two critical CVEs, CVE-2026-32194 and another, to address these vulnerabilities.
  • This vulnerability could lead to unauthorized access and control over critical systems, potentially allowing attackers to manipulate data or disrupt services. Given the scale of Bing's infrastructure, successful exploitation could have widespread implications for users and services relying on Bing's image processing capabilities. Immediate remediation is essential to prevent potential exploitation.
  • An attacker could submit a specially crafted SVG file to Bing's image search, leveraging the vulnerability to execute commands with elevated privileges on the underlying servers.
  • Watch next: Monitor for updates from Microsoft regarding the patching of the identified CVEs and any related security advisories.
Key terms
  • SVG - Scalable Vector Graphics, a file format for vector graphics that can include executable code.
  • CVE - Common Vulnerabilities and Exposures, a list of publicly disclosed cybersecurity vulnerabilities.
Defender focus
  • Implement monitoring for unusual command executions on Bing's image processing servers.
  • Review access logs for any unauthorized submissions or command executions related to SVG files.
  • Ensure that all systems are updated with the latest patches from Microsoft to mitigate the identified vulnerabilities.
Do now
  • Patch all affected systems immediately with the updates provided by Microsoft.
  • Conduct a security audit of the image processing workflows to identify any additional vulnerabilities.
  • Enhance monitoring and alerting for any suspicious activity related to SVG file submissions.
Watch next: Monitor for updates from Microsoft regarding the patching of the identified CVEs and any related security advisories.
