One-Click Microsoft 365 Copilot Flaw Could Have Let Attackers Steal Emails, Files, and MFA Codes

A critical vulnerability in Microsoft 365 Copilot could allow attackers to exfiltrate sensitive data with a single click, posing immediate risks to enterprise security.

Who should care: IT security teams managing Microsoft 365 environments and organizations utilizing Microsoft 365 Copilot.
Attack path: An attacker can send a malicious link that appears legitimate, leading users to click and inadvertently expose their data. This single-click method simplifies the attack process and increases the likelihood of successful exploitation.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: A flaw in Microsoft 365 Copilot could let attackers steal sensitive information easily.
microsoft vulnerability data exfiltration
  • Researchers discovered a vulnerability in Microsoft 365 Copilot that allows attackers to exfiltrate emails, calendar details, and files with a single click on a malicious link. This vulnerability, termed SearchLeak, results from a combination of three separate bugs. The link appears legitimate as it directs users to a real microsoft.com domain, bypassing traditional security measures.
  • This vulnerability could lead to significant data breaches, compromising sensitive information and potentially allowing unauthorized access to accounts. Organizations using Microsoft 365 Copilot are at heightened risk, as attackers can exploit this flaw to gather critical data without raising alarms.
