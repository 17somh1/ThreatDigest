LeakNet ransomware uses ClickFix, Deno runtime in stealthy attacks

LeakNet ransomware's new techniques could lead to significant data breaches and operational disruptions for affected organizations.

Who should care: IT security teams and incident response units in organizations using JavaScript or TypeScript applications should be particularly vigilant.
Attack path: Attackers gain leverage by exploiting vulnerabilities in web applications to execute the ClickFix technique, allowing them to deploy the Deno-based malware loader undetected.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: LeakNet ransomware is using advanced methods to infiltrate corporate networks, posing a serious threat to data security.
ransomware cve cloud
  • The LeakNet ransomware group has adopted the ClickFix technique for initial access into corporate environments. They are also utilizing a malware loader based on the Deno runtime, which is designed for JavaScript and TypeScript applications.
  • These methods enhance the stealth of the ransomware attacks, making detection more challenging for security teams. If successful, these attacks can lead to severe data loss and operational downtime, impacting business continuity.
  • Attackers gain leverage by exploiting vulnerabilities in web applications to execute the ClickFix technique, allowing them to deploy the Deno-based malware loader undetected.
  • Watch next: Monitor for any updates from security vendors regarding patches for vulnerabilities that could be exploited by ClickFix.
Key terms
  • ClickFix - a technique used to gain initial access to systems by exploiting vulnerabilities in web applications.
  • Deno - a secure runtime for JavaScript and TypeScript that allows developers to run scripts and applications.
Defender focus
  • Implement monitoring for unusual web application behavior that could indicate exploitation attempts.
  • Review and update security controls around JavaScript and TypeScript applications to mitigate potential vulnerabilities.
  • Conduct threat hunting exercises focused on detecting the use of Deno in your environment.
Do now
  • Enhance monitoring for ClickFix-related attack patterns.

Why this matters now: Useful context for teams tracking where attacker pressure is building.

phishing malware kakaotalk

  • This method of attack allows for rapid propagation of malware within organizations, potentially leading to widespread infection. The reliance on KakaoTalk for communication increases the risk of sensitive information being compromised.
