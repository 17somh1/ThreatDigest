Laser Attack Resets Tangem Wallet Passwords on Cards That Can't Be Patched

A laser attack can reset Tangem wallet passwords, allowing attackers to steal cryptocurrency without any prior access.

Who should care: Tangem wallet users and security teams managing cryptocurrency assets should be particularly concerned.
Attack path: An attacker can leverage a laser to reset the wallet's password, gaining control over the funds stored in the wallet without needing any prior credentials.
Why this matters now: A live patch or vendor fix gives defenders something concrete to validate.
In plain English: A new laser attack can compromise Tangem wallet cards, enabling unauthorized access to funds.
crypto vulnerability security
  • Researchers demonstrated that a precisely timed laser pulse can reset the password of Tangem crypto wallet cards. This attack does not require the old password or a backup card, allowing the attacker to take full control of the wallet.
  • This vulnerability poses a significant risk to Tangem wallet users, as it allows attackers to easily access and transfer funds without any user intervention. The inability to patch the cards means that affected users remain at risk indefinitely.
  • An attacker can leverage a laser to reset the wallet's password, gaining control over the funds stored in the wallet without needing any prior credentials.
  • Watch next: Monitor for any updates from Tangem regarding mitigation strategies or new wallet designs.
Key terms
  • Laser Attack - a method of using focused light to manipulate electronic components.
  • Tangem Wallet - a type of cryptocurrency wallet stored on a card that uses secure chips.
Defender focus
  • Implement monitoring for unauthorized access attempts on Tangem wallets.
  • Advise users to transfer funds to more secure wallets until a fix is available.
  • Review physical security measures for devices that interact with Tangem wallets.
Do now
  • Alert all Tangem wallet users about the vulnerability and recommend immediate action.
  • Investigate the feasibility of transferring funds to alternative wallets.
  • Stay updated on any official communications from Tangem regarding this issue.
