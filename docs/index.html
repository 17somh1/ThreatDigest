CISA Adds One Known Exploited Vulnerability to Catalog

The addition of CVE-2026-28318 to the KEV Catalog signals an urgent need for organizations to address a critical vulnerability that is actively being exploited.

Who should care: Federal Civilian Executive Branch agencies and any organizations using SolarWinds Serv-U should prioritize remediation efforts.
Attack path: An attacker could exploit this vulnerability to consume excessive resources, potentially leading to denial of service or unauthorized access. This type of vulnerability is commonly targeted by cyber actors.
Why this matters now: KEV status shortens the acceptable patch window.
In plain English: CISA has identified a new vulnerability in SolarWinds Serv-U that is currently being exploited, requiring immediate attention.
cve solarwinds vulnerability
  • CISA has added CVE-2026-28318, an Uncontrolled Resource Consumption Vulnerability in SolarWinds Serv-U, to its Known Exploited Vulnerabilities Catalog. This addition is based on evidence of active exploitation, highlighting the vulnerability's significance.
  • This vulnerability poses a substantial risk to federal networks and potentially to other organizations as well. Timely remediation is crucial to prevent exploitation, which could lead to unauthorized access or service disruption.
  • An attacker could exploit this vulnerability to consume excessive resources, potentially leading to denial of service or unauthorized access. This type of vulnerability is commonly targeted by cyber actors.
  • Watch next: Monitor for updates from CISA regarding additional vulnerabilities added to the KEV Catalog and any recommended remediation timelines.
Key terms
  • CVE - Common Vulnerabilities and Exposures, a list of publicly disclosed cybersecurity vulnerabilities.
  • KEV - Known Exploited Vulnerabilities, a catalog of vulnerabilities that are actively being exploited in the wild.
Defender focus
  • Implement monitoring for unusual resource consumption patterns in SolarWinds Serv-U.
  • Ensure that all instances of SolarWinds Serv-U are updated to the latest version to mitigate this vulnerability.
  • Review and prioritize remediation plans for vulnerabilities listed in the KEV Catalog.
Do now
  • Patch SolarWinds Serv-U immediately to address CVE-2026-28318.

  • Enhance monitoring for any signs of exploitation related to this vulnerability.
