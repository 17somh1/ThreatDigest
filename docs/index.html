Who should care: Development and security teams managing the Claude Extension and users of the Chrome browser should be particularly vigilant.
Attack path: An attacker can exploit this vulnerability by crafting a malicious website that injects prompts into the Claude Extension, effectively controlling it without any user consent.
Why this matters now: Useful context for teams tracking where attacker pressure is building.
In plain English: A flaw in the Claude Extension allows for zero-click prompt injections from any website.
vulnerability chrome xss
  • Researchers identified a vulnerability in the Claude Google Chrome Extension that enables zero-click prompt injection. This means that simply visiting a malicious website could allow an attacker to send commands to the assistant without any user action.
  • This vulnerability can lead to unauthorized actions being taken on behalf of users, potentially compromising sensitive information or executing harmful commands. The lack of user interaction required for exploitation increases the risk of widespread attacks.
